# Payment and Billing Support Docs

These documents provide users of the **QuickBite** food delivery app with clear, step-by-step guidance on payments, common billing issues, and resolutions. QuickBite supports secure, convenient payments through trusted third-party gateways (e.g., Razorpay, PayU, or similar in India). All transactions are encrypted and compliant with PCI-DSS standards.

Use the in-app "Help" section or "Orders" tab to access receipts, payment history, and report issues. For urgent problems, tap "Chat with Support" on any order screen.

## 1. Payment Methods Guide

QuickBite offers multiple secure options tailored for users in India and other regions.

**Supported Payment Methods (as of February 2026)**:

- **UPI** (Unified Payments Interface): Instant transfers via apps like Google Pay, PhonePe, Paytm, BHIM.
- **Credit/Debit Cards**: Visa, Mastercard, RuPay, American Express (processed via secure gateways).
- **Digital Wallets**: Paytm Wallet, Amazon Pay, Mobikwik, Freecharge.
- **Net Banking**: Major Indian banks (e.g., HDFC, ICICI, SBI, Axis).
- **Cash on Delivery (COD)**: Available for select orders/restaurants (limited to low-value or first-time users to reduce risks).
- **Other Options**: International cards (for tourists), gift cards/vouchers, and platform credits (from refunds/promotions).

**How to Add or Manage Payment Methods**:

1. Go to Profile > Payment Methods.
2. Tap "Add New" and follow prompts (e.g., scan UPI QR, enter card details securely).
3. Set a default method for faster checkout.
4. Remove or edit saved methods anytime—QuickBite does not store full card details (tokenized via gateway).

**Tips for Smooth Payments**:

- Ensure sufficient balance/funds before checkout.
- For UPI: Keep your UPI PIN ready; enable notifications for OTPs.
- Update expired cards promptly to avoid failed recurring promotions.
- Use saved methods to skip re-entering details.

## 2. Billing Disputes Resolution Guide

Billing disputes include unauthorized charges, double billing, incorrect amounts, promo code failures, or unrecognized transactions.

**Common Dispute Types and Resolutions**:
Use this table for quick reference:

| Issue Type                   | Description                                                            | Steps to Resolve                                                                                                                                      | Expected Timeline                                    |
| ---------------------------- | ---------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------- |
| **Unauthorized Charge**      | Transaction not made by you (possible fraud/hacked account).           | 1. Freeze card via bank.<br>2. Report in-app: Orders > Select order > Report Issue > "Unauthorized Charge".<br>3. Provide transaction ID and details. | Immediate block; refund 3–7 days after verification. |
| **Double Billing**           | Same order charged twice.                                              | Check payment history in app/bank statement. Report via in-app chat with screenshots.                                                                 | Auto-detect often; refund 1–5 days.                  |
| **Incorrect Amount Charged** | Wrong total (e.g., extra fees, tax miscalculation, promo not applied). | Review breakdown at checkout. Report: Orders > "Report Billing Issue" > Upload screenshot of checkout vs. charged amount.                             | Refund/credit within 24–48 hours.                    |
| **Promo Code Failure**       | Discount not applied despite eligibility.                              | Re-check terms (e.g., min order value). Report with order ID and promo code used.                                                                     | Credit issued if valid; usually instant or 1 day.    |
| **Unrecognized Transaction** | Charge appears but no order placed.                                    | Verify in "Payments" tab. If fraudulent, follow unauthorized charge steps.                                                                            | Investigation 3–10 days; full refund if confirmed.   |

**General Dispute Process**:

1. **Gather Evidence**: Screenshots of order confirmation, payment receipt, bank statement, and any error messages.
2. **Report Promptly**: Use in-app reporting within 30 days of transaction (earlier is better for faster resolution).
3. **Resolution Options**:
   - Full refund to original payment method.
   - Platform credit (usable on future orders, often with bonus).
   - Escalation to human support if AI-assisted review needs more info.
4. **Chargeback via Bank**: As a last resort (e.g., if QuickBite can't resolve). Note: Chargebacks may lead to account restrictions if deemed abusive.

**Prevention Tips**:

- Enable transaction alerts from your bank/UPI app.
- Review order summary carefully before confirming payment.
- Use strong passwords and enable 2FA on your QuickBite account.

## 3. Tax and Invoice Explanations Guide

Understand what you're paying for transparency.

**Breakdown of Charges on Your Bill**:

- **Food Subtotal**: Price of items as listed on restaurant menu.
- **Taxes & GST**: Government-mandated (e.g., 5% on essentials, 18% on restaurant services in India).
- **Delivery Fee**: Distance-based + dynamic (surge during peak hours).
- **Service Fee**: QuickBite platform charge (typically 10–20%) for tech, support, and operations.
- **Packaging Fee**: Charged by restaurant for eco-friendly or special packaging.
- **Small Order Fee**: Applied if subtotal below threshold (e.g., ₹150) to cover costs.
- **Convenience/Tip**: Optional tip for delivery partner (100% goes to them).
- **Promo Discounts/Credits**: Applied automatically if eligible.

**Accessing Invoices/Receipts**:

1. Go to "Orders" > Select completed order > Tap "View Receipt" or "Download Invoice".
2. Receipt includes:
   - Order ID and timestamp.
   - Itemized breakdown.
   - Payment method used.
   - Transaction ID (useful for disputes).
3. Email receipts sent automatically to registered email; check spam if missing.

**Tax-Related Queries**:

- GST invoices available for business users (request via support with GSTIN).
- Taxes calculated per restaurant policy and local laws—no hidden additions post-checkout.

**Important Notes**:

- All fees displayed transparently before payment confirmation.
- Refunds include proportional taxes/fees where applicable.
- For international users: Currency conversion fees may apply via your card issuer.

These docs empower QuickBite users to handle 90% of payment-related queries independently. If issues persist, our in-app support (AI + human escalation) is available 24/7. For policy updates or region-specific changes, check the app footer links.
